# Jogo da Velha Online #
Uma implementação clássica do Jogo da Velha (Tic-Tac-Toe) desenvolvida para partidas online. Simples, leve e focado na experiência do usuário.

## Sobre o Projeto
Este projeto permite que dois jogadores se enfrentem em tempo real. O objetivo foi criar uma interface limpa e uma lógica de jogo sólida.

## 🛠️ Tecnologias Utilizadas
Frontend: Pygame.

Comunicação: Socket.io com tcp.

## 🕹️ Como Jogar

Inicie a partida: O primeiro jogador começa como sol e o segundo como chuva.

Vença o jogo: Alinhe três símbolos iguais na horizontal, vertical ou diagonal.

Empate: Se o tabuleiro encher sem um vencedor, temos a famosa "Velha"!

## Instalação e Execução
Caso queira jogar rode o client.bat primeiro e depois o tela.bat
